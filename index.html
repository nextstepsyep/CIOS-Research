Original Contributions

Four metrics not present in existing SGPBL literature

These four metrics are CIOS's original theoretical contribution to IEEE. No prior publication combines a monetized skill assessment, a passive-to-active activation measure, a community resolution rate, and an analog telemetry index in a single validated framework.

01
Novel Metric 01
Contribution Value Growth Rate
CV_rate = (CV_T₁ − CV_T₀) / Curriculum_days

CV_T = Σ(Have/Use_domain × $CV_domain)
Domain weights: $10 · $20 · $30 · $40 · $50 · $100
Converts learning gain into economic value at the individual level using pre-calibrated, game-validated skill values. No existing SGPBL study attaches monetized weights to a validated skill assessment instrument — making this a direct bridge between educational measurement and economic impact reporting.
CIOS: DB #20 · DB #58 · DB #45
02
Novel Metric 02
Potential Value Conversion Rate
PV_conv = (PV_T₀ − PV_T₁) / PV_T₀

PV = Σ(Have/Didn't Use × $CV_domain)
Measures passive → active skill activation
Measures the curriculum's success at activating passive skills — those possessed but not yet applied — into active contribution. The distinction between skill possession and skill application is unique to the CoD Skills Self-Assessment and cannot be replicated by any other SGPBL framework currently in the literature.
CIOS: DB #20 T₀/T₁ delta · Domain surveys
03
Novel Metric 03
Stakeholder's Dilemma Resolution Rate
SDRR = Dilemmas_Resolved / Dilemmas_Initiated × 100

Resolved: Status="Resolved" ∧
CE_Zone_score_T₁ > CE_Zone_score_T₀
Requires community-level evidence confirmation
Measures whether the community problem was actually solved — not just whether participants learned something about it. Links individual learning outcomes to documented socio-economic change in a specific geographic location. No existing PBL assessment includes a community problem resolution rate as a primary dependent variable.
CIOS: DB #10 · DB #8 · DB #9 · DB #22
04
Novel Metric 04
Decision Revision Index
DRI = SIP_Revisions / Total_SIP_Activities

Revision: Plan/Design revised after
Coordinate/Evaluate phase begins
Analog telemetry for physical serious games
An analog telemetry measure for PBL adaptive expertise that does not require digital game infrastructure — applicable to physical serious games and hybrid PBL formats. Captures when participants change course in response to new evidence, stakeholder feedback, or changed conditions: the hallmark of engineering-grade problem-solving competency.
CIOS: DB #27 SIP Activities · Revision fields
Statistical Analysis

IEEE-standard calculations built into the CIOS output layer

CIOS Layer 7 (Impact) and Layer 8 (Reports) produce the structured datasets required for each statistical test below. No external data extraction is needed — every formula draws directly from named CIOS database fields.

MANOVA
Multivariate Analysis of Variance
Compares CoD SGPBL vs. control across all six domain scores, SIMS subscales, and community impact measures simultaneously — controlling for Type I error inflation across multiple dependent variables.
Λ = Wilks' Lambda (overall effect)
η² = Partial eta squared (effect size)
Source: DB #20 × 6 domains · DB #20a SIMS
Normalized Gain ⟨g⟩
Hake Learning Gain (1998)
Assesses improvement while controlling for baseline knowledge across all six Achievement Domains. Enables cross-domain and cross-CE Zone comparison standardized to entry level.
⟨g⟩ = (%Post − %Pre) / (100 − %Pre)
Low <0.30 · Medium 0.30–0.70 · High ≥0.70
Source: DB #20 T₀ and T₁ per domain
Cohen's d
Effect Size
Calculates the magnitude of the CoD SGPBL intervention relative to pooled standard deviation. SGPBL benchmark: d = 0.46–0.85. Expected Domain 6 result: d ≥ 0.80 due to high U-score baseline in CE Zone participants.
d = (M_post − M_pre) / SD_pooled
SD_pooled = √[(SD_pre² + SD_post²) / 2]
Source: DB #20 domain Have/Use scores
SIMS · SDI
Situational Motivation Scale
Guay et al. (2000) 16-item validated instrument across four subscales: Intrinsic Motivation, Identified Regulation, External Regulation, Amotivation. Administered at T₀ and T₁ via CIOS DB #20a.
SDI = 2(IM) + (IR) − (ER) − 2(AM)
Higher SDI = more self-determined motivation
Source: DB #20a Motivation Assessment
SROI
Social Return on Investment
Monetizes Eight Capital gains against total program investment. CoD SGPBL target: ≥4.0× SROI. The CV Growth Rate feeds the SROI numerator with a novel skill-activation economic valuation component unavailable in other frameworks.
SROI = Total_Social_Value / Total_Investment
Benchmark: Community ed 2.1–3.4× · WFD 3.5–6.2×
Source: DB #46 · DB #45 · DB #55 · DB #58
SEM
Structural Equation Modeling
Maps how CoD SGPBL components drive engagement and subsequent learning. Tests the causal pathway: game engagement → motivation → skill acquisition → community impact → economic value.
CFI ≥ 0.95 · RMSEA ≤ 0.06 · SRMR ≤ 0.08
Path: DB#27→DB#20a→DB#20→DB#8→DB#46
Source: All CIOS Layer 4, 5, 7 databases

Study Design

Quasi-experimental, multi-site, longitudinal

Each CE Zone operating identical 60-database CIOS architecture enables cross-community comparison at a scale unavailable to single-site SGPBL studies. Every site produces structurally identical datasets — making meta-analysis and network effect measurement possible within a single research cycle.

Design
Quasi-experimental
Pre-test / post-test with control group. Experimental: full BFYF curriculum (Sections 1–9). Control: comparable service-learning without CoD SGPBL framework.
Sample
N ≥ 30 per CE Zone
Minimum 3 CE Zones for multi-site comparison. IEEE publication standard. Longitudinal span: Phase I (1–2 wks) + Phase II (2–4 wks) + Phase III (3–6 months).
Instruments
6 validated surveys
Skills Self-Assessment (6 domain surveys) + SIMS motivation scale (16 items, 4 subscales) + CE Zone Assessment (condition scoring 0–100) + TRU Capability Profile.
Primary DVs
Domain Have/Use scores
Six Achievement Domain scores (T₀ and T₁) + TRU Capability Score delta + SIMS SDI change + CE Zone condition score change + SROI ratio pre/post.
Novel DVs
4 original metrics
CV Growth Rate · PV Conversion Rate · Stakeholder's Dilemma Resolution Rate (SDRR) · Decision Revision Index (DRI). Not present in existing SGPBL literature.
Analysis
MANOVA · SEM · Cohen's d
MANOVA (multivariate), ANOVA (cross-site), paired t-test (within group), ⟨g⟩ normalized gain, Cohen's d effect size, SEM path modeling, SROI calculation.
Research Question
